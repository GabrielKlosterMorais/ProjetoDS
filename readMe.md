# DESAFIO DS
## Projeto de ds focado em por em pratica o CRUD

**Tematica do projeto** a ideia de gatos veio de um jogo chamado Mewgenics que esta em alta por, o projeto funciona an logica de cada gato tem 1 pessoa mas uma pessoa pode ter gato X e outra pode não ter esse gato mas pode ter o gato Y e Z

**Para oque o projeto serve** o projeto serve para um banco de dados de consulta de colecionadores

**Como Rodar na minha maquina** primeiro instale o "node_modules" usando o "npm i" no terminal powershell e então usar "npm start", para vc usar o banco precisa usar o postman usando as urls (http://localhost:8080/api/people) para coisas menos expecificas tipo colocar alguem no banco e pegar todas as pessoas da tabela, mas para usar coisas mais especificas como deletar alguem, consultar os dados de alguem especifico deve usar (http://localhost:8080/api/people/"Id da pessoa").
